# Solar Panel Recycling Platform: India’s Recycling and Collection Gap

**Problem Statement:**  
End-of-life solar panels are often stored, dumped, or sold to untrained scrap dealers, causing environmental pollution and wastage of valuable materials. People also do not know the remaining life of their panels or where to recycle them safely.

**Purpose:**  
This repository is a **demo version for hackathon expert track evaluation**.  
It contains frontend mockups, dashboards, and demo API structure. Core business logic is intentionally excluded for IP protection.

## Demo Features
- Pickup request form (frontend mockup)
- Dashboard views for admin, industry, and users
- API endpoints with mock responses
- Flowcharts and diagrams illustrating process
